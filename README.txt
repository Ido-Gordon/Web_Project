
# Construction Calculator - אתר מחשבוני בנייה

האתר נועד לספק פתרון פרקטי ונוח לקבלנים, מנהלי עבודה ובעלי מקצוע בתחום הבנייה, באמצעות שלושה מחשבונים אינטראקטיביים:

## 🧱 מחשבונים באתר:
1. **מחשבון גדרות** - מחשב את מספר העמודים, המרווחים ביניהם, והשארית האפשרית בהתאם לאורך גדר נתון וטווח רווחים רצוי.
2. **מחשבון בלוקים** - מחשב את כמות הבלוקים הנדרשת לבניית קיר לפי גובה, אורך וסוג הבלוק שנבחר (כולל מחירים ומידות).
3. **מחשבון דק** - מחשב את כמות העץ הנדרשת לפי רוחב ואורך הדק המבוקש ורוחב קרש העץ.

## 📄 עמודים מרכזיים:
- `index.html` – עמוד הבית, ניווט לכל אחד מהמחשבונים.
- `about.html` – עמוד מידע כללי על המערכת.
- `fence-calculator.html` – מחשבון גדרות.
- `block-calculator.html` – מחשבון בלוקים.
- `deck-calculator.html` – מחשבון דק.

## ⚙️ קבצי קוד:
- `styles.css` – קובץ עיצוב רספונסיבי ואחיד לכל הדפים.
- `calc.js` – מכיל את כל פונקציות החישוב והדינמיקה של המחשבונים.
- `JavaScrip.js` – סקריפטים כלליים נוספים (למשל התחברות).

## 🛠 טכנולוגיות:
- HTML5
- CSS3 (כולל Flexbox, Media Queries)
- Bootstrap 5
- JavaScript וניהול DOM

הפרויקט מותאם גם לדסקטופ וגם לנייד, עם ממשק נוח, אינטראקטיבי ונגיש לכל משתמש מקצועי בתחום.
